Exercise: Scoreboard

Instructions

We will be creating a simple scoreboard application using JavaScript. The HTML and CSS has already been done for you.

Tasks

  1. Add a link to your own custom JS file.
  2. Get all the interactive Elements on the page and save them in variables.
  3. Setup a click event listeners on the +5 and -5 point buttons. Change the innerHTML of the score display appropriately.
  4. Setup a click event listener on the Set Score button. Set the innerHTML of the score display to the score entered in the text box. You'll need to get the .value of the text box.

    Remember: An Element's value property is a string. You're going to need to use a little parseInt() magic to convert it so you can do the maths.

Refactoring tip

Always practice the principle of DRY. Create a separate function to update the DOM and make the changes to the score display. Instead of having to write your DOM manipulation logic over and over in each event handler.

Bonuses

Lets make it more resilient.

  1. Create a check in your code to make sure the score will not go negative.
  2. Check to make sure the input value is a number. Do nothing if it's not a number.

This exercise draws the following skills

  • events
  • event listeners
  • concatenation
