D3 (Data-Driven Document)

D3.js (for Data-Driven Documents) is a JavaScript library for manipulating documents based on data and producing dynamic, interactive data visualizations in web browsers. D3 helps you bring data to life using HTML, SVG, and CSS. Lesson #7 - learning OBJECTS

DOM is the skeleton of a webpage, it's a tree of objects
  1. Objects is like the most powerful datatype because it can encapsulate lot of information which is grouped in 2 groups: property and methods
  2. For Property, each must be in one of these datatypes:
    Number, String, Array, Boolean or another object.
  3. For Method, it is actually function
    but stoed inside an object. And must be called thru Object.
  4. So it can be nested in multiple levels
  5. to delcare an object, each property is encapsulated inside a pair of curly brackets then each property is actually a pair of STRING values:
    Property Name and Property Value where each is encapsulated by a double quote
    then concatenated thru a colon ":"
  6. Then each pair of property is separated by a comma.
  7. Each property name must be a string
  8. Each property value can be any data type
  9. In addition to contain multliple different ATTRIBUTES for different data types, an object can also contain functions (or called 'Method')

There are 3 different categories of objects
  • Host Environment - Window, Document, navigator, location
  • Professional Public Utilities Library - JSON, D3, Math
  • User-defined - Real life objects like Bank, Movie

it solved the problem from other language that you have to declare it before using it) but what is done by HOISTING ?
There are 2 reads of the HTML/CSS/JS code by the parser of the browser, first read is to pick up all statements like functions or variables. So when the parser reads the 2nd round, such functions and variables are already noticed so they can be used without any issue.

Other materials taught include :
  1. data-types: String, Number, Boolean, Array and Objects
    • For Array, it can contain any types of data but each member
      must have the same data type of other members inside the array
      therefore,the whole array will contain multiple items of the same type of data. ARRAY is an object with property 'length'
      and 3 methods: pop(), push() and forEach()
    • For Object, it can contain any types of data but each member
      (can be property or method) can have different data type from other
      members inside an object
    • Conversion amongst different data types are allowed thru
      different methods from each declared data variable because
      each declared data variable itself is an instance of an object
      for instance: toString(), JSON.ParseStr(), ParseInt() and ParseFloat()
  2. Conditional Flow-control statement: IF-ELSE, IF-ELSEIF-ELSE
  3. Loop : Array.forEach(callback fx) + traditional for-loop statement
